== EXPRESS-G diagrams

The diagrams in this annex correspond to the EXPRESS schemas specified in this
part of ISO 10303. The diagrams use the EXPRESS-G graphical notation for the
EXPRESS language. EXPRESS-G is defined in ISO 10303-11.
